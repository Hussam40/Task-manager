📝 Task Manager - Python Console App
A simple and interactive console-based task management system built with Python. It allows users to input their daily tasks, track their completion status step-by-step, and view a summary of their progress.

📌 Features
Interactive Prompts: Interactively asks for the completion status of each task entered.

Categorization: Automatically separates tasks into Finished and Unfinished lists.

Input Validation: Ensures the user only enters valid responses (Yes or No) and prevents errors.

Progress Summary: Displays a clean summary of completed and pending tasks at the end of the session.

📜 How it Works
Input Tasks: The program prompts you to enter your tasks separated by a comma (e.g., Task 1, Task 2, Task 3).

Track Status: For each task, you will be asked if you finished it (Yes or No).

View Progress: Once completed, the program will ask if you want to view your progress, displaying the lists of finished and unfinished tasks.
